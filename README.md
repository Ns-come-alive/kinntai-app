# 勤怠管理アプリ

ナイトワーク向けのシンプルな Web ベース勤怠管理アプリケーションです。

## 機能

### キャスト側
- 名前タップでログイン（パスワード不要）
- 出勤 / 退勤の打刻（複数回可）
- 同伴出勤の打刻
- 遅刻時の理由選択（交通機関の遅延・体調不良・その他）
- 月別勤怠履歴（出勤日数・稼働時間・遅刻時間・欠勤日数）

### 管理者側
- シフト登録・管理
- 本日の全キャスト勤怠状況の一覧
- 当欠チェック（シフト登録済み＆打刻なしを一括処理）
- キャスト別月間サマリー（タブ切り替え）
- 全キャスト勤怠履歴の閲覧・CSV エクスポート

### 業務ロジック
- **営業日:** 20:00 ～ 翌 09:00 を1日として扱う
- **遅刻判定:** シフト開始時刻を過ぎた出勤は「遅刻」
- **同伴出勤の猶予:** シフト開始から1時間以内なら遅刻にならない
- **遅刻時間:** 30分刻みで切り上げ計算（例: 1分遅刻 → 0.5時間）

## 技術スタック

- **バックエンド:** Python / Flask
- **データベース:** SQLite（ファイルベース、セットアップ不要）
- **フロントエンド:** HTML / CSS / JavaScript

## セットアップ

```bash
# リポジトリのクローン
git clone https://github.com/<your-username>/kintai-app.git
cd kintai-app

# 依存パッケージのインストール
pip install -r requirements.txt

# アプリの起動
python app.py
```

ブラウザで http://localhost:5000 にアクセスしてください。

## 初回利用

1. キャスト名をタップしてログイン
2. ダッシュボードで「出勤する」または「同伴出勤」で打刻

### 管理者ログイン
- ログイン画面下部の「管理者ログイン」を開く
- パスワード: `admin1234`（本番環境では変更してください）
